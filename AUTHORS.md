## Acknowledgments

### Academic References

The mathematical foundations of this library are based on the work of many researchers in extreme value theory, probability theory, and statistics. Key references include:

- **de Haan, L. and Ferreira, A.** (2006). *Extreme Value Theory: An Introduction*. Springer.
- **Hill, B. M.** (1975). A Simple General Approach to Inference About the Tail of a Distribution. *The Annals of Statistics*, 3(5), 1163-1174.
- **Pickands III, J.** (1975). Statistical Inference Using Extreme Order Statistics. *The Annals of Statistics*, 3(1), 119-131.
- **Embrechts, P., Klüppelberg, C., and Mikosch, T.** (1997). *Modelling Extremal Events*. Springer-Verlag.
- **Beirlant, J., Goegebeur, Y., Segers, J., and Teugels, J.** (2004). *Statistics of Extremes: Theory and Applications*. John Wiley & Sons.

### Software Inspirations

This project draws inspiration from several excellent statistical software packages:

- **R's `evd` package** by Alec Stephenson
- **SciPy's stats module** by the SciPy developers
- **NumPy** for numerical computing patterns
- **Statsmodels** for statistical modeling approaches

### Educational Resources

Special recognition to educational materials that influenced the pedagogical approach:

- MIT 18.650 (Statistics for Applications) course materials
- Stanford CS229 (Machine Learning) probability review
- "All of Statistics" by Larry Wasserman
- "An Introduction to Statistical Learning" by James, Witten, Hastie, and Tibshirani

## Contributing

We welcome contributions from researchers, practitioners, and students interested in heavy-tailed distributions and extreme value theory. Please see [CONTRIBUTING.md](CONTRIBUTING.md) for guidelines.

### How to Be Listed

Contributors will be added to this file when they make significant contributions such as:

- **Code contributions**: New distributions, algorithms, or substantial improvements
- **Documentation**: Major documentation improvements or examples
- **Testing**: Comprehensive test suites or validation studies
- **Mathematical contributions**: New theoretical insights or algorithm improvements
- **Bug reports and fixes**: Identifying and fixing important issues

### Recognition Criteria

- **First contribution**: Name added to contributors section
- **Significant contribution**: More detailed description of contributions
- **Ongoing contributions**: Recognition as a core contributor or co-maintainer
