## Acknowledgments

### Inspiration and References

HeavyTails draws inspiration from several excellent libraries and resources:

**Python Libraries:**

- **SciPy** - Reference implementations for validation
- **NumPy** - Numerical computing patterns (not a dependency)
- **statsmodels** - Statistical modeling approaches

**R Packages:**

- **POT** (Peaks Over Threshold) by Mathieu Ribatet
- **evir** by Alexander McNeil and team
- **tea** by Jan Beirlant and team

**Academic Works:** The implementation is based on rigorous mathematical foundations from:

1. **Embrechts, P., Kl√ºppelberg, C., & Mikosch, T. (1997).** _Modelling Extremal Events for Insurance and Finance._ Springer.

2. **Resnick, S. I. (2007).** _Heavy-Tail Phenomena: Probabilistic and Statistical Modeling._ Springer.

3. **de Haan, L., & Ferreira, A. (2006).** _Extreme Value Theory: An Introduction._ Springer.

4. **Hill, B. M. (1975).** "A Simple General Approach to Inference About the Tail of a Distribution." _Annals of Statistics_, 3(5), 1163-1174.

5. **Pickands, J. (1975).** "Statistical Inference Using Extreme Order Statistics." _Annals of Statistics_, 3(1), 119-131.

## How to Contribute

We welcome contributions of all kinds! Here's how you can help:

### Code Contributions

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Submit a pull request
